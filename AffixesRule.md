# 写在最前
为了帮助快速记忆词缀赋予的属性，制定本"词缀简述缩写规则"；
因游戏中属性顺序和模拟器的排序不一致，所以本规则以游戏为准。

简述要求：属性直观、描述统一、排序统一。

# 简述缩写规则

## 总排序规则

### 游戏中装备属性排序 参考：
(点伤/暴伤/吸血/伤害类型/伤害转化/种歧增伤/三维HPMP/%三维HPMP/OADA/%OADA/回复/护甲/速度/伤害抗性/元素抗性/副抗/CDR/盾挡类属性/其他对怪物效果)

### 普通词缀 规则：
(暴伤/吸血/伤害类型/伤害转化/种歧增伤/三维HPMP/%三维HPMP/OADA/%OADA/护甲/速度/主副抗/CDR/盾挡类属性)

### 召唤词缀 规则：
(召唤/战宠名/%普通词缀%)

## 主要 伤害类型/抗性类型 排序：
    物理-穿刺-火焰-冰冷-闪电-酸毒-活力-虚化-混乱-元素-流血
               └─ 元素 ─┘

### 主要 伤害类型/抗性类型 缩写：
物理伤害/创伤伤害/物理抗性      = 物
穿刺伤害/穿刺抗性               = 穿
火焰伤害/燃烧伤害/火焰抗性      = 火
冰冷伤害/霜燃伤害/冰冷抗性      = 冰
闪电伤害/电击伤害/闪电抗性      = 电
酸性伤害/毒素伤害/毒酸抗性      = 酸
活力伤害/活力衰减/活力抗性      = 活
虚化伤害/虚化抗性               = 虚
混乱伤害/混乱抗性               = 混
元素伤害/元素抗性               = 元
流血伤害/流血抗性               = 血

## 伤害类型转化参考"主要伤害类型"排序和缩写
如：物理伤害 转换为 元素伤害    = 物转元

## 三维HPMP/%三维HPMP 排序
    体格-灵巧-精神-HP-MP-%体格-%灵巧-%精神-%HP-%MP

### 三维HPMP/%三维HPMP 缩写 (如括号总字数过长，请缩写此项)
体格    = 体
灵巧    = 灵
精神    = 精
HP      = HP
MP      = MP
%体格   = %体
%灵巧   = %灵
%精神   = %精
%HP     = %HP
%MP     = %MP

## OADA/%OADA 排序
    攻击能力-防御能力-%攻击能力-%防御能力

### OADA/%OADA 缩写
攻击能力    = OA
防御能力    = DA
%攻击能力   = %OA
%防御能力   = %DA

## 护甲 排序
    护甲提高-护甲吸收提高

### 护甲 缩写
护甲提高        = 甲
护甲吸收提高    = 甲吸

## 速度 排序
    攻击速度-施法速度-移动速度-所有速度

### 速度 缩写
攻击速度    = AS
施法速度    = CS
移动速度    (这个不写入简述，BD通常不重要)
所有速度    = TS

## 主副抗 主抗排序和缩写参考"主要伤害类型/抗性类型"
### 最后加"抗"以区分伤害 (其余副抗不写)
    主抗-副抗(眩晕抗性-禁锢抗性-石化抗性-冻结抗性-减速抗性-反弹抗性)

### 副抗缩写
眩晕抗性    = 晕
禁锢抗性    = 禁锢
石化抗性    = 石
冻结抗性    = 冻
减速抗性    = 减速
反弹抗性    = 反
